# Sintaxe Básica e Variáveis

## Declarando variáveis

```go
var nome string = "Enzo"
var idade int = 17
```

## Declaração curta

```go
nome := "Enzo"
```

## Tipos primitivos

| Tipo | Exemplo |
|---|---|
| string | "Olá" |
| int | 10 |
| float64 | 5.5 |
| bool | true |

## Constantes

```go
const PI = 3.14
```

## Comentários

```go
// Comentário de uma linha
```

## Print

```go
fmt.Println("Texto")
```

!!! note
Go possui tipagem estática.
